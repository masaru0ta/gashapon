# 機能一覧

|  #  | カテゴリ | 機能名 | 概要 |
|-----|---------|--------|------|
| 1-1 | マップ | マップ表示 | 地形データの管理と描画。18種の地形タイルでマップを構成する |
| 1-2 | マップ | 占領システム | GB・都市・コロニー・補給基地・スペースベースの占領処理。占領による拠点化・CP収入・回復効果 |
| 2-1 | 移動 | ユニット移動 | ユニットタイプ（一般・水陸両用・艦船）ごとの移動コスト計算と移動処理 |
| 2-2 | 移動 | 搭載システム | 艦船へのユニット搭載・発進。搭載中は移動・戦闘不可、毎ターン20%回復 |
| 2-3 | 移動 | マップ射撃 | 射程内の敵ユニットへのマップ上射撃。EN消費、固定ダメージ |
| 3-1 | 戦闘 | 戦闘画面 | ユニット遭遇時のアクションバトル画面遷移。HP・ENの表示と管理 |
| 3-2 | 戦闘 | 戦闘地形効果 | 戦闘画面での地形効果（障害物・慣性・重力/浮力）の適用 |
| 3-3 | 戦闘 | 武器システム | 15種の武器（ビームライフル、マシンガン等）の実装。A/B/AB操作、威力・弾速・装填数 |
| 3-4 | 戦闘 | 戦闘アイテム | 戦闘中にランダム出現するアイテム（力ドリンク・高速足袋・招き猫・縛り札） |
| 3-5 | 戦闘 | 障害物キャラクター | 特定地形で出現する障害物（岩・アステロイド・木）の生成と当たり判定 |
| 4-1 | ユニット | ユニットデータ管理 | 全ユニットのパラメータ管理（CP・TN・MV・SP・EN・武装）。ジオン・連邦・共通・ボス |
| 5-1 | 生産 | ユニット生産 | 生産拠点（GP/GB）でのユニット生産。CP消費とターン数による完成管理 |
| 6-1 | 回復 | 回復システム | 補給基地（100%）・都市（10%/ターン）・艦船搭載（20%/ターン）での回復処理 |
| 7-1 | ゲーム進行 | ターン管理 | ターンの進行処理。ターン開始時の回復・生産完了・CP収入などの自動処理 |
| 7-2 | ゲーム進行 | 勝利条件判定 | 敵本拠地占領・敵全滅・ターン制限による勝敗判定 |
| 8-1 | ゲームモード | 1P vs CPU | プレイヤー対コンピュータの対戦モード |
| 8-2 | ゲームモード | 1P vs 2P | プレイヤー同士の対戦モード |
| 8-3 | ゲームモード | シナリオモード | ガンダムシリーズ各作品モチーフのシナリオプレイ。シナリオ別勝利条件 |
| 9-1 | AI | CPU AI | 敵指揮官（マクベ・バスク・ハマーン・シャア）ごとの行動パターンと戦闘AI |
